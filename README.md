# Gismeteo Rainmeter skin
Виджет прогноза погоды от Gismeteo для Rainmeter.
- ![Почасовой прогноз со светлым фоном виджета над тёмным участком](/screenshots/Gismeteo%20chrome%20Hourly_LightBg.jpg))
- ![Недельный прогноз с тёмным фоном виджета над светлым участком](/screenshots/Gismeteo%20chrome%20Weekly_DarkBg.jpg)
- ![Полный прогноз без фона виджета над тёмным участком](/screenshots/Gismeteo%20chrome%20All_NoBg.jpg)

## Возможности:
* Режимы отображения прогноза погоды: 1) на текуший день с показом изменений на каждые 3 часа текущих суток; 2) на ближайшие 7 дней, начиная с текущего; 3) оба предыдущих варианта, но без общего прогноза на текущий день.
    - В прогнозе на текущие сутки заголовок текущего времени будет отображаться зелёным цветом шрифта, прошедших - серым, а для будущих - белым.
    - В прогнозе на неделю дата дней попадающих на субботу и воскресенье будут выделяться розовым цветом шрифта.
    - При наведении на иконку погоды показывает дополнительную информацию:

![Подсказка о состоянии погоды на 5 июля](/screenshots/Gismeteo%20chrome%20Weekly.jpg)

![Подсказка о состоянии погоды на 21 час](/screenshots/Gismeteo%20chrome%20Hourly.jpg)
* Позволяет ввести код нужного места без необходимости редактировать файлы. Достаточно кликнуть левой кнопкой мыши по названию текущего выбраного места чтобы появилась форма ввода.

![Ввод кода места, для которого будет показываться прогноз погоды](/screenshots/Gismeteo%20chrome%20City.jpg) 
* Позволяет переключать фон виджета кнопкой без необходимости смены виджета или его перезагрузки (*кнопка переключения фона видна под блоком дополнтительной информации о текущей погоде, на первых 3 скриншотах показаны подсказки, всплывающие при наведении на доступные варианты выбора*).
    - Также с помощью кнопок можно быстро переключаться между режимами отображения информации о погоде, чтобы не использовать контекстное меню программы (*сама кнопка расположена левее кнопки переключения фона виджета*).
    - Кнопки отображаются только при наведении курсора на виджет.
* Кликом по описанию текущей погоды можно обновить информацию на виджете не обновляя сам виджет через контекстное меню. При наведении курсора на это описание будет показано последнее обновление данных виджетом.

![Информация об обновлении данных](/screenshots/Gismeteo%20chrome%20Updated.jpg)
* За иконкой текущей погоды отображается в виде круговой диаграммы длительность ночи голубым цветом и индикацией текущего времени светло-зелёной меткой, что позволяет увидеть сколько осталось до восхода или заката.
    - Под этой иконкой и информацией о текущей температуре воздуха также отображается информация о длительности дня, времени восхода и заката в текстовом виде.
* В момент обновления данных из интернета отображается значок правее иконки погоды, между описанием погоды и тепературой воздуха. Это для визуализации процесса обновления данных, чтобы видно было идёт ли ещё получение информации с сервера.
    - Если будут проблемы с доступом к серверу данных, то вместо описания погоды будет текст "*ConnectError*", а при возникновении проблемы с парсингом основного блока данных - "*RegExpError*".

## Существующие проблемы
* У виджета есть проблема с отображением длительности дня, если выбирать локации, где он длится более суток (*например, северный и южный полюса, это коды 10997 и 14743, соответственно*). У меня для полюсов показывает отрицательные значения для длительности дня.
    - Это потому что используется параметр передающий только время для календарных суток, а не передаваемый UNIX-таймштамп для этого же. Возможно это будет мною когда-нибудь исправлено, но гарантировать это не могу.

## Возможные проблемы
* Текущая дата берется из системы и из-за этого возможно неправильное раскрашивание дат в недельном прогнозе. Как получать точный день недели из получаемых данных простым способом я пока не придумал.

## Использование сторонних ресурсов
* Пиктограммы для обозначения длительности дня, восхода и заката, а так же для дополнительной информации о текущей погоды - это глифы шрифта **[Weather Icons](https://github.com/erikflowers/weather-icons/)** от [Erik Flowers](https://github.com/erikflowers).
    - 6 из 7 значков хорошо подошли, для температуры воды уже спорно, а для геомагнитной активности подходящий не нашёл, но это и неважно, т.к. Gismeteo на текущее время всегда передаёт 0 и я убрал эту строку за ненадобностью. Если знаете более подходящие символы у шрифтов в бесплатном доступе, то всегда можете мне посоветовать альтернативу (*я бы не отказался от других вариантов для всех иконок, кроме восхода и заката: те вполне себе нормально смотрятся*).
* Фоны были взяты возможно откуда-то из других каких-то виджетов Rainmeter, но это не точно: они просто хранились у меня несколько лет в папке со скинами для Rainmeter и вспомнить откуда они были взяты не получается.
