# Gismeteo Rainmeter skin
Виджет прогноза погоды от Gismeteo для Rainmeter.

![Скриншоты двух режимов отображения с обоими вариантами фона виджета](/screenshots/Gismeteo_chrome_1.0.3_27.06.2026.jpg)

### Возможности:
* Показывает прогноз погоды либо на текуший день с показом изменений на каждые 3 часа текущих суток, либо на ближайшие 7 дней, начиная с текущего. В режиме показа прогноза на текущие сутки заголовок текущего времени будет отображаться зелёным цветом, прошедших - серым, а для будущих - белым.
При наведении на иконку погоды показывает дополнительную информацию:

![Прогноз на текущий день с подсказкой о состоянии погоды на 29 июня](/screenshots/weekly-icon_tooltip.jpg)

![Прогноз на текущий день с подсказкой о состоянии погоды на 12 часов](/screenshots/hourly-icon_tooltip.jpg)
* Позволяет ввести код нужного места без необходимости редактировать файлы. Достаточно кликнуть левой кнопкой мыши по названию текущего выбраного места чтобы появилась форма ввода.

![Ввод кода места, для которого будет показываться прогноз погоды](/screenshots/weekly-city.jpg) 
* Позволяет переключать фон виджета кнопкой без необходимости смены виджета или его перезагрузки (*кнопка переключения фона видна на самом первом скриншоте*).
* Кликом по описанию текущей погоды можно обновить информацию на виджете не обновляя сам виджет через контекстное меню. При наведении курсора на это описание будет показано последнее обновление данных виджетом.

![Информация об обновлении данных](/screenshots/weekly-refresh.jpg)
